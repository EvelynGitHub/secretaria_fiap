# Por onde começar:

## Back end

Pré-requisitos: Docker, make, git

```bash
# clonar repositório
git clone https://github.com/EvelynGitHub/secretaria_fiap

# entrar na pasta do projeto back end
cd secretaria_fiap

# executar o projeto
make build
make up

# parar o projeto
make down

```
